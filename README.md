# 💬 SNSアプリ

## 概要

PHP・MySQLを使用して開発した投稿型のSNSアプリです。

ログイン後、テキスト投稿や画像投稿を行うことができ、投稿数やいいね数に応じてランキング上位を目指せるWebアプリです。

## 工夫した点

### 💬 投稿機能

ログイン中のユーザー情報をもとに、投稿内容をデータベースへ保存できるようにしました。

投稿にはユーザー名やプロフィール画像を紐づけ、誰が投稿した内容か分かりやすく表示しています。

---

### 🖼 画像投稿

テキストだけでなく画像も投稿できるようにし、SNSらしい見た目になるよう工夫しました。

---

### 🏆 ランキング機能

投稿数やリアクション数に応じてランキングを表示し、ユーザーが継続して利用したくなる仕組みを意識しました。

---

### 👤 ユーザー管理

PHPとMySQLを使用して、会員情報・ログイン状態・プロフィール画像を管理しています。

また、マイページやアカウント情報変更、退会機能も実装しました。

---

## 苦労した点

ログイン中のユーザー情報と投稿データを紐づける部分に苦労しました。

セッション情報を利用してユーザーIDを取得し、投稿テーブルに保存することで、投稿者情報を正しく表示できるようにしました。

また、画像投稿ではアップロードした画像の保存先や表示パスがずれることがあり、フォルダ構成とパス指定を見直して修正しました。
